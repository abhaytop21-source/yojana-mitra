# 🌾 Yojana Mitra – Maharashtra Farmer Welfare Portal

Yojana Mitra is a web platform designed to help farmers easily discover, understand, and access government schemes available in Maharashtra. The platform simplifies scheme information, eligibility criteria, required documents, and application guidance in a user-friendly interface.

## 🚀 Features

### 🌱 Government Scheme Directory

* Browse government schemes for farmers
* View scheme details and benefits
* Eligibility criteria information
* Required documents list

### 🔍 Smart Scheme Finder

* Filter schemes based on:

  * District
  * Crop Type
  * Land Size
  * Irrigation Status
  * Age
  * Bank Account & KYC Status

### 📚 Farmer Guides

* PMFBY Claim Process
* KYC & Bank Linking Guide
* Soil Testing Guide
* Government Application Procedures

### ❓ Farmer Q&A Section

Categories include:

* Pests & Diseases
* Irrigation & Water Management
* Crop Insurance
* Agricultural Loans
* Organic Farming
* Market Prices
* Government Documentation

### 🌐 Bilingual Support

* English
* Marathi

### 📱 Responsive Design

* Mobile Friendly
* Tablet Friendly
* Desktop Friendly
